# Canton Yard Care Client-Facing Demo

This is a polished static website demo for `cantonyardcare.club`.
It is designed to look like a real local lawn care website that could be shown to a potential client.

## Files

- `index.html` — page content and structure
- `styles.css` — all design and responsive styling
- `script.js` — mobile menu and footer year
- `CNAME` — GitHub Pages custom domain file

## What to change before going live

Open `index.html` and update:

1. Phone number: `(770) 555-0198`
2. Email address: `hello@cantonyardcare.club`
3. Form action: `https://formspree.io/f/your-form-id`
4. Pricing, if needed
5. Reviews, once real reviews are available
6. Service area, if the provider serves different cities

## GitHub Pages setup

1. Create a new GitHub repository.
2. Upload these files directly to the root of the repository.
3. Go to **Settings → Pages**.
4. Choose the `main` branch and root folder.
5. Add the custom domain: `cantonyardcare.club`.
6. Confirm your GoDaddy DNS records point to GitHub Pages.

## Notes

This site intentionally avoids putting code on one line.
The HTML, CSS, and JavaScript are formatted so they are readable and easy to edit.
